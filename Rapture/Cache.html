Overview

This class is abstract.

Subclass and override #resolve, #cache, #remove, and #each to implement a custom Cache.

Note:

See DefaultCache as an example of an “in memory” Cache implementation.

Heavy inspiration for abstract caching has been drawn from z64's discordcr cache refactor

Direct Known Subclasses

DefaultCache, NullCache
